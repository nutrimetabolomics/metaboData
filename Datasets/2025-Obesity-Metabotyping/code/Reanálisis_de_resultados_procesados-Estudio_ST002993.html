Informaci√≥n del estudio

Este estudio se basa en el proyecto depositado en MetabolomicsWorkbench:

https://www.metabolomicsworkbench.org/data/DRCCMetadata.php?Mode=Project&ProjectID=PR001863

EL resumen del proyecto en el repositorio es el siguiente:

La obesidad es el resultado de una interacci√≥n entre la predisposici√≥n gen√©tica y factores ambientales como la dieta, la actividad f√≠sica, la cultura y el nivel socioecon√≥mico. Los tratamientos personalizados ser√≠an ideales, por lo que es necesario identificar caracter√≠sticas individuales que permitan mejorar la eficacia de las terapias. Por ejemplo, alteraciones gen√©ticas en la v√≠a leptina-melanocortina pueden provocar casos raros de obesidad grave de inicio temprano.

La metabol√≥mica tiene el potencial de distinguir entre un estado saludable y uno obeso; sin embargo, diferenciar subgrupos de individuos dentro del espectro de la obesidad sigue siendo un reto. El an√°lisis factorial puede integrar caracter√≠sticas cl√≠nicas y moleculares de los pacientes a partir de diversas fuentes, permitiendo una subclasificaci√≥n precisa.

Este estudio presenta un flujo de trabajo para identificar metabotipos, especialmente en contextos donde los estudios cl√≠nicos convencionales no logran categorizar adecuadamente a los pacientes. Se estudiaron 110 ni√±os con obesidad (IMC > +2 SDS), genotipados para nueve genes implicados en la v√≠a leptina-melanocortina (CPE, MC3R, MC4R, MRAP2, NCOA1, PCSK1, POMC, SH2B1 y SIM1) y dos genes de receptores de glutamato (GRM7 y GRIK1). De ellos, 55 presentaban variantes raras heterocigotas y 55 no ten√≠an variantes.

Se recogieron datos antropom√©tricos y de laboratorio cl√≠nico rutinario, y las muestras de suero se procesaron para an√°lisis metabol√≥micos no dirigidos utilizando GC-q-MS, CE-TOF-MS y U(H)PLC-QTOF-MS/MS en modos de ionizaci√≥n positiva y negativa. Tras el procesamiento de se√±ales y la multialineaci√≥n, se aplicaron an√°lisis estad√≠sticos multivariantes y univariantes para evaluar la asociaci√≥n entre los rasgos gen√©ticos y los datos metabol√≥micos y cl√≠nicos.

Ni la presencia de variantes gen√©ticas raras heterocigotas ni los datos cl√≠nicos rutinarios permitieron identificar subgrupos dentro de los datos metabol√≥micos. Para descubrir subtipos metabol√≥micos, se aplic√≥ an√°lisis factorial1 sobre una matriz compuesta construida a partir de las cinco plataformas2 anal√≠ticas. Se identificaron seis factores y tres metabotipos distintos. Se observaron diferencias sutiles pero claras en los l√≠pidos circulantes y en la sensibilidad a la insulina, lo que abre la posibilidad de personalizar el tratamiento en funci√≥n de la clasificaci√≥n del paciente dentro de estos subtipos de obesidad.

La identificaci√≥n de metabotipos en contextos cl√≠nicos presenta desaf√≠os debido a la influencia de m√∫ltiples variables no controladas sobre los fenotipos metab√≥licos. Sin embargo, esta estrategia demuestra su potencial para identificar subconjuntos de pacientes con diagn√≥sticos cl√≠nicos similares pero con condiciones metab√≥licas distintas. Este enfoque pone de relieve la aplicabilidad del an√°lisis factorial para el estudio de metabotipos en diversos escenarios cl√≠nicos.

Resultados

En el estudio se usaron varias plataformas anal√≠ticas:

  • AN004913 : LC-MS positivo
  • AN004914 ‚Üí LC-MS negativo
  • AN004916 ‚Üí CE-MS
  • AN004917 ‚Üí GC-MS

Segn la tecnolog√≠a, la primera columna se indica como Mass_RT y Mass_MT que identifican cada pico metabol√≥mico combinando la masa (m/z) con el tiempo.

Mass_RT se usa en LC-MS y GC-MS (tiempo de retenci√≥n), mientras que Mass_MT se usa en CE-MS (tiempo de migraci√≥n).

Ambos permiten identificar de forma √∫nica cada feature y pueden unificarse como Feature_ID.

Filtrado

Si hay NAs eliminaremos las muestras que superen un porcentaje dado.

if (anyNA(data_matrix)) {

  message("Se han detectado valores NA. Aplicando filtrado e imputaci√≥n...")

  # Filtrado: eliminar features con m√°s del 20% de NA
  na_prop <- rowMeans(is.na(data_matrix))
  data_matrix_filt <- data_matrix[na_prop <= 0.2, ]

  # Imputaci√≥n: reemplazar NA con el m√≠nimo valor positivo de cada muestra
  data_matrix_imp <- apply(data_matrix_filt, 2, function(x) {
    x[is.na(x)] <- min(x[x > 0], na.rm = TRUE)
    x
  })

} else {
  message("No se han detectado valores NA. Continuando sin filtrado ni imputaci√≥n.")
  data_matrix_imp <- data_matrix
}

## An√°lisis factorial m√∫ltiple (MFA)

library(FactoMineR)
library(factoextra)

# 1. Matriz de expresi√≥n
X <- t(assay(se, "data_matrix_norm"))

# 2. Preparar data.frame y a√±adir variable cualitativa
X_df <- as.data.frame(X)
X_df$obesity_type <- colData(se)$obesity_type

# 3. Preparar info de grupos
platforms <- rowData(se)$platform
vars_by_platform <- split(rownames(se), platforms)
X_df <- X_df[, c(unlist(vars_by_platform), "obesity_type")]

group_sizes <- c(sapply(vars_by_platform, length), 1)
group_types <- c(rep("c", length(vars_by_platform)), "n")  # √∫ltima es cualitativa
group_names <- c(names(vars_by_platform), "obesity")

# 4. Ejecutar MFA
res_mfa <- MFA(
  base = X_df,
  group = group_sizes,
  type = group_types,
  name.group = group_names,
  num.group.sup = length(group_sizes),  # √∫ltima posici√≥n es suplementaria
  graph = FALSE
)
fviz_screeplot(res_mfa, addlabels = TRUE, ylim = c(0, 50))
# 5. Visualizar individuos (muestras)
fviz_mfa_ind(
  res_mfa,
  habillage = "obesity_type",
  addEllipses = TRUE,
  palette = c("red", "blue"),  # Colores personalizados
  label = "none",              # No mostrar etiquetas de muestras
  repel = FALSE,
  title = "MFA - Muestras coloreadas por tipo de obesidad"
)
# Extraer informaci√≥n de variables y grupos
